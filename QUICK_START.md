# 🚀 دليل البدء السريع

## ✅ ما تم إنجازه

تم بناء صفحة احترافية متكاملة لعرض تفاصيل منتج **كرواسون فرنسي فاخر** بمواصفات تصميمية عصرية وسهلة الاستخدام.

---

## 📁 الملفات الرئيسية

### 1. **`lib/main.dart`**
- نقطة الدخول الرئيسية للتطبيق
- إعدادات الـ Theme والألوان
- التوجيه للصفحة الرئيسية

### 2. **`lib/pages/product_detail_page.dart`**
- صفحة تفاصيل المنتج الرئيسية
- إدارة حالة الكمية (State Management)
- جميع العناصر والتفاعلات

### 3. **`DOCUMENTATION.md`**
- شرح شامل لكل عنصر في الصفحة
- المميزات التقنية
- نصائح للتخصيص

### 4. **`SPECIFICATIONS_DETAILS.md`**
- جدول تفصيلي بجميع الألوان والأحجام
- قياسات دقيقة لكل عنصر
- نصائح التطوير

### 5. **`SPECIFICATIONS.csv`**
- ملف CSV يمكن فتحه في Excel
- جداول منظمة لجميع المواصفات
- سهل البحث والتصفية

| 🛒 الأزرار | إضافة للسلة + شراء فوري | النهاية |

## 🎮 الميزات التفاعلية

### ➕ زيادة الكمية
- اضغط على زر `+` الأحمر/الذهبي
- يزيد الرقم بمقدار 1
- لا حد أقصى للزيادة

### ➖ تقليل الكمية
- اضغط على زر `-` الأحمر/الذهبي
- ينقص الرقم بمقدار 1
- **الحد الأدنى: 1** (لا ينقص أقل من ذلك)

### 🛒 إضافة إلى السلة
- يعرض رسالة: "تمت إضافة X كرواسون إلى السلة"
- الرسالة تختفي بعد ثانيتين
- اللون: بني داكن

### ⚡ شراء فوري
- يعرض رسالة: "سيتم الشراء الفوري الآن"
- الرسالة تختفي بعد ثانية

### ♡ إضافة للمفضلة
- اضغط على أيقونة القلب في الأعلى
- يعرض رسالة التأكيد

### ← الرجوع
- اضغط على السهم في الأعلى يساراً
- العودة للصفحة السابقة

## 🔧 تخصيص المشروع

### 🎨 تغيير الألوان

افتح `lib/main.dart` و `lib/pages/product_detail_page.dart` وعدّل:

```dart
// قبل
const Color(0xFFFDFBF7)  // بيج دافئ

// بعد
const Color(0xFF12345A)  // اللون الجديد
```

### 🖼️ تغيير صورة المنتج

في `lib/pages/product_detail_page.dart`:

```dart
// قبل
'https://images.unsplash.com/photo-1555507036-ab1f4038808a?w=500&h=500&fit=crop'

// بعد
'https://your-image-url.com/croissant.jpg'
```

### 📝 تغيير المعلومات

في `lib/pages/product_detail_page.dart`:

```dart
// اسم المنتج
'كرواسون فرنسي فاخر'  // غيّر هنا

// السعر
'45.00'  // غيّر هنا

// الوصف
'كرواسون فرنسي مخبوز بـ...'  // غيّر هنا
```

### 📏 تغيير الأحجام

ابحث عن:
```dart
fontSize: 28      // حجم الخط
borderRadius: 20  // حواف دائرية
```

## 📊 استخدام الملفات

### 💾 فتح ملف CSV في Excel

1. **Windows:**
   - انقر نقراً مزدوجاً على `SPECIFICATIONS.csv`
   - أو افتح Excel → ملف → فتح → اختر الملف

2. **Mac:**
   - انقر نقراً مزدوجاً على `SPECIFICATIONS.csv`
   - أو افتح Numbers/Excel

3. **Google Sheets:**
   - اذهب إلى Google Drive
   - Upload الملف
   - فتحه مباشرة في Sheets
