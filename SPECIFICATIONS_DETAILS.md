## 📐 المقاييس والأحجام

### 🔤 أحجام الخطوط (Font Sizes)

| العنصر | الحجم | الوزن (FontWeight) | الاستخدام | مثال |
|--------|-------|------------------|-----------|--------|
| عناوين المنتج | 28 px | bold | اسم المنتج | "كرواسون فرنسي فاخر" |
| الأسعار | 32 px | bold | عرض السعر الأساسي | "45.00" |
| العملة | 20 px | w600 | رمز العملة | "ج.م" |
| الأزرار | 18 px | w600 | النصوص على الأزرار | "إضافة إلى السلة" |
| العناوين الثانوية | 16 px | w600 | "الكمية"، "المدة" | "الكمية" |
| النصوص العادية | 15 px | normal | وصف المنتج | "كرواسون فرنسي مخبوز بـ..." |
| التفاصيل الصغيرة | 14 px | normal | عدد التقييمات | "(127 تقييم)" |
| العناصر الصغيرة | 12-13 px | normal/w600 | تسميات البطاقات | "🕐 المدة" |

### 📏 الأبعاد والمسافات (Dimensions & Padding)

| العنصر | الحجم | الملاحظات |
|--------|-------|----------|
| **الصورة** | 35% من ارتفاع الشاشة × 85% من عرض الشاشة | تتكيف مع حجم الجهاز |
| **الحواف الناعمة (BorderRadius)** | | |
| - الصورة | 20 px | حواف دائرية سلسة |
| - البطاقات والعناصر | 12 px | حواف ناعمة قياسية |
| - الأزرار | 12 px | تناسق مع البطاقات |
| **المسافات (Padding)** | | |
| - هوامش الصفحة | 20 px أفقي | من جانبي الشاشة |
| - المسافة بين العناصر | 12-32 px | تبعاً للأهمية |
| - الحشو الداخلي (Padding) | 16 px | داخل البطاقات |
| **الارتفاعات** | | |
| - أزرار الإجراء | 56 px | متوافق مع معايير Material |
| - أزرار الكمية | 45 px | متناسقة مع الحقل الأوسط |
| - حقل الكمية الأوسط | 60 px عرض، 45 px ارتفاع | مربع شبه متوازن |

## 🎬 الحركات والتفاعلات (Interactions)

| التفاعل | الحدث | الاستجابة |
|---------|-------|-----------|
| زر + | الضغط | زيادة الكمية بـ 1 |
| زر − | الضغط | تقليل الكمية بـ 1 (الحد الأدنى: 1) |
| "إضافة إلى السلة" | الضغط | عرض SnackBar بلون بني (#3E2723) |
| "شراء فوري" | الضغط | عرض رسالة تأكيد |
| ♡ المفضلة | الضغط | عرض رسالة إضافة للمفضلة |
| ← الرجوع | الضغط | العودة للصفحة السابقة |

## 📱 التجاوب (Responsiveness)

| حجم الشاشة | التأثير |
|-----------|--------|
| صغيرة (< 360px) | تمرير سلس، تناسب تلقائي |
| متوسطة (360-600px) | العرض الطبيعي الموصى به |
| كبيرة (600-900px) | نفس التخطيط، مسافات أفقية أكبر |
| كبيرة جداً (> 900px) | نفس التخطيط، قد تحتاج حاوية مركزية |

## ✨ نصائح التطوير

1. **تغيير الألوان:** استخدم Find & Replace لتغيير جميع الألوان دفعة واحدة
2. **الأحجام المرنة:** استخدم `MediaQuery` للأحجام الديناميكية
3. **الصور:** استبدل URL الصورة بصور منتجاتك الخاصة
4. **الترجمة:** النصوص العربية مدعومة بالكامل
5. **الأداء:** استخدم `const` قدر الإمكان للثوابت
